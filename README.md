# BOTLEAGUE Landing Page

A modern responsive landing page built with React, Vite, and Tailwind CSS based on the provided Figma design.

## Features

* Responsive UI
* Hero Section
* Competitions & Events
* User Journey
* About BOTLEAGUE
* Categories
* Competition Disciplines
* League Benefits
* Join the Ecosystem
* Sponsors & Social Media Footer

## Tech Stack

* React.js
* Vite
* Tailwind CSS

## Installation

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
